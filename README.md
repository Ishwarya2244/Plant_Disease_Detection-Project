# 🌿 Plant Disease Detection using AI

## 📌 Overview

This project uses **Deep Learning (CNN)** to detect plant diseases from leaf images.
It helps in identifying diseases at an early stage, which can support better crop management and productivity.

---

## 🚀 Features

* 🌱 Detects plant diseases from leaf images
* 🤖 Uses Convolutional Neural Network (CNN)
* ⚡ Fast and accurate predictions
* 📊 Visualizes results using graphs

---

## 🛠️ Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* OpenCV
* Matplotlib
* Jupyter Notebook

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/your-username/plant-disease-detection.git

2. Navigate to the folder:
   cd plant-disease-detection

3. Install dependencies:
   pip install -r requirements.txt

4. Open Jupyter Notebook:
   jupyter notebook

5. Run the notebook file:
   plant_disease_detection.ipynb
